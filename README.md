# Healthcare Data Analytics Platform

## Project Overview

This project demonstrates an end-to-end Healthcare Data Analytics workflow using:

- Python
- SQL
- Power BI
- AWS

The objective is to analyze healthcare patient records and generate insights related to treatment costs, diagnoses, hospital performance, and patient demographics.

---

## Dataset

The dataset contains healthcare patient information including:

- Patient ID
- Age
- Gender
- Diagnosis
- Hospital
- Admission Date
- Discharge Date
- Treatment Cost

File:

data/patient_data.csv

---

## SQL Analysis

The project includes SQL queries to analyze:

- Total treatment cost by hospital
- Average treatment cost by diagnosis
- Number of patients by diagnosis
- Most expensive treatments
- Average patient age by diagnosis

File:

sql/analysis_queries.sql

---

## Python Analysis

Python and Pandas are used for:

- Data loading
- Data cleaning
- Cost analysis
- Diagnosis analysis
- Summary statistics

Notebook:

notebooks/healthcare_analysis.ipynb
